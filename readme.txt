This is the source code of the Firefox extension YCS (Youtube Comment Search) - v. 1.1.12
It is simply unzipped from the publically distributed .xpi on Firefox add-on site.

The "original" folder contains simply the files unpacked from the .xpi file
The "beautified" folder contains the same files run through various code beautifiers, to make it more readable.
No other modifications have been made.
